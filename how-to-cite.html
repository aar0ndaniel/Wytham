Citation Guidelines

How to Cite metis

Help researchers cite metis in theses and academic papers
Website: https://metis.emend.it.com

Why Cite metis?

If you use metis for your research, teaching, or thesis work, please cite it so that others can find and use the tool, and so that the effort to build and maintain metis is visible to the academic and research communities.

Download Citation Files

Use one of these files directly in your citation management tool:

BibTeX for LaTeX, Overleaf, TeXShop
RIS for Mendeley, Zotero, RefWorks
Endnote for EndNote, ProCite
CSL JSON for Citation Style Language tools

APA 7th Edition

Use this format for in-text citations and reference lists:

Metis. (2026). Metis: Free PLS-SEM desktop software for academic researchers (Version 0.0.1) [Computer software]. https://metis.emend.it.com

BibTeX

Use this format if you are using BibTeX or similar bibliography managers:

@software{metis2026,
  title = {Metis: Free PLS-SEM Desktop Software for Academic Researchers},
  author = {{Metis}},
  year = {2026},
  version = {0.0.1},
  url = {https://metis.emend.it.com},
  note = {Computer software}
}

Software Version and URL

  • Version: 0.0.1 (beta)
  • Website: https://metis.emend.it.com
  • Release Year: 2026

Update the version number if you are citing a newer release of metis.

Also Cite SEMinR and Methodological Sources

Metis uses R and SEMinR under the hood to perform all PLS-SEM calculations. To give credit to the statistical foundations of your analysis, please also cite:

  • SEMinR — the R package that powers metis analysis
  • Relevant methodological sources — the PLS-SEM papers and techniques you used in your study

This ensures that credit is properly attributed to all the tools and methods that made your research possible.

Example Reference Section

Here's how your full reference section might look when you cite metis alongside SEMinR and a PLS-SEM methodology paper:

Hair, J. F., Ringle, C. M., & Sarstedt, M. (2013). Partial least squares structural equation modeling: Rigorous applications, better results. Long Range Planning, 46(1–2), 1–12.

Metis. (2026). Metis: Free PLS-SEM desktop software for academic researchers (Version 0.0.1) [Computer software]. https://metis.emend.it.com

Ray, S., Danks, N. M., & Dawson, J. (2021). seminr: Domain-specific language for building and estimating structural equation models (Version 2.2.0) [R package]. https://CRAN.R-project.org/package=seminr
