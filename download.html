Installation note

Because metis is new and not yet code-signed, Windows may show a security warning during installation. If this appears, choose More info, then Run anyway. Please download metis only from the official website.

Basic installation steps

  1. Download the installer.
  2. Open the file.
  3. Follow the installation prompts.
  4. Launch metis from your desktop or Start Menu.

metis is currently available for Windows. Linux and macOS versions are in development.
