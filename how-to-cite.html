Citation Guidelines

How to Cite metis

Help researchers cite metis in theses and academic papers
Website: https://metis.emend.it.com

Why Cite metis?

If you use metis for your research, teaching, or thesis work, please cite it so that others can find and use the tool, and so that the effort to build and maintain metis is visible to the academic and research communities.

Download Citation Files

APA 7th Edition

Use this format for in-text citations and reference lists:

Metis. (2026). Metis: Free PLS-SEM desktop software for academic researchers (Version 0.0.1) [Computer software]. https://metis.emend.it.com

BibTeX

Use this format if you are using BibTeX or similar bibliography managers:

@software{metis2026,
  title = {Metis: Free PLS-SEM Desktop Software for Academic Researchers},
  author = {{Metis}},
  year = {2026},
  version = {0.0.1},
  url = {https://metis.emend.it.com},
  note = {Computer software}
}
