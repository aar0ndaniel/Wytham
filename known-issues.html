Known Issues

A short, practical list of current notes for metis users.

Current Notes

  • Metis is currently available for Windows.
  • Linux and macOS versions are in development.
  • Some advanced modules may still be refined as the app grows.
  • Large datasets may take longer depending on the user's computer.
  • The Windows installer may show a security warning because code signing is not yet completed.
  • Users should verify important research results before using them in final publications.

Windows Security Warning

Because Metis is new and not yet code-signed, Windows may show a security warning during installation. If this appears, choose More info, then Run anyway. Please download Metis only from the official website.

Report Something Broken

For crashes, installation failures, wrong results, freezes, or error messages, please report the issue so it can be tracked and fixed.
